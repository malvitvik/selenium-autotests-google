# Google autotests

Google Autotests. Usage: Selenium (WebDriverManager) + Cucumber + Junit4 + log4j + maven 

## Installation

Create IDEA progect from version control > Git. Paste link to the current repository in it.

## Usage

Run autotests with JUnit4 or 

```
mvn clean test
```
